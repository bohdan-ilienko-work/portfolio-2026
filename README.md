# Bohdan Ilienko Portfolio (2026)

This repository contains the personal portfolio website of **Bohdan Ilienko**, Full Stack Engineer.

The project was built to present engineering experience, stack, projects, and delivery approach in one place for recruiters, teams, and clients.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Three.js / React Three Fiber

## Run locally

```bash
npm install
cp .env.example .env.local
npm run dev
```

Required env variable:

- `NEXT_PUBLIC_SITE_URL` (example: `https://your-domain.com`)

## i18n and SEO routing

- Locales are configured in `i18n/routing.ts` (`en`, `uk`, `ru`, `pl`, `de`, `es`).
- Locale-prefixed routes are generated by App Router under `app/[locale]`.
- Messages are split by section and locale in:
  - `messages/{locale}/common.json`
  - `messages/{locale}/hero.json`
  - `messages/{locale}/grid.json`
  - `messages/{locale}/projects.json`
  - `messages/{locale}/testimonials.json`
  - `messages/{locale}/experience.json`
  - `messages/{locale}/cv.json`
  - `messages/{locale}/footer.json`

### Add a new locale

1. Add locale code to `locales` in `i18n/routing.ts`.
2. Create `messages/{new-locale}/` and copy all section JSON files with the same keys.
3. Add translations to each file.
4. Rebuild the app to confirm locale routes and metadata alternates are generated correctly.
